# Project: sdlc

## 🔒 Publishing & privacy policy (สำคัญ)
- **ห้าม publish หรือทำให้เนื้อหาของโปรเจกต์นี้เป็น public บน claude web / Artifacts** โดยไม่ได้รับอนุญาตชัดเจนจากผู้ใช้ก่อน "ทุกครั้ง"
- Artifacts ที่สร้างต้องคงสถานะ **private เท่านั้น** — ห้ามแชร์ลิงก์สาธารณะหรือแนะนำให้แชร์
- ไฟล์ในโปรเจกต์ (spec, source code, รายงานวิจัย, เดโม) ถือเป็น **ข้อมูลภายใน** ห้ามส่งออกไปยังบริการภายนอก
- ก่อนเรียกเครื่องมือที่เผยแพร่เนื้อหาออกนอกเครื่อง ต้องถามยืนยันก่อนเสมอ
